C++ Class

A class in C++ follows basic Object Oriented Programming the difference between C++ and other languages (Python, Java, C#) it doesn't have a garbage collector. This can lead to memory issues when not properly handled.

This is an example of a class in C++ that has public/private variables and methods 


                    class Goblin {
                        public: //Makes the variables public which means system can use 
                                // them without getters/setters
                            float health = 100; //basic variable that every object (goblin) 
                                                // created will start with
                            int damage = 10;
                        private:
                            std::string item; //Private variable which means only this class can see
                                         // and modify it

                        private: //Makes the methods/vars private so that only the class can use/see these
                            string item; //Private variable which means only this class can see
                                         // and modify it
                            //Since the goblin calculates the health inside the class 
                            // the rest of the system doesn't need to see this method
                            void CalculateHealth(int damage) { 
                                health = health - damage;
                            }

                        public: //For public methods
                            //This is a default constructor which means it takes no variables
                            // and uses default values
                            Goblin() {
                                damage = 10;
                                health = 100;
                            }
                            //This is an overloaded constructor which is called when you create a goblin 
                            //It takes paramaters to be able to change the classes variables
                            Goblin(int damage) {
                                //Using this->damage lets the compiler know that you are talking
                                // to the classes variable and not the one thats being passed through
                                this->damage = damage;
                            }
                            //This is a deconstructor, whenever this object is deleted it calls this
                            //method
                            ~Goblin() {
                                std::cout << "Goblin has been destroyed" << std::endl;
                            }
                            //This is a setter which allows you to change private variables 
                            void setItem(std::string item) {
                                this->item = item;
                            }
                            //This is a getter which allows a system to get the classes variable value 
                            // without getting the actual variable
                            std::string getItem() {
                                return item;
                            }
                            //This is a method that the class will retain when created 
                            void recieveDamage(int damage) {
                                std::cout << "Goblin took damage" << std::endl;
                                CalculateHealth(damage);
                            }
                    };

In this example we use the goblin class by creating an object and using its methods then deleting the object using manuel garbage collection 


                    int main() {
                        // Create the goblin class
                        Goblin* bob = new Goblin(10);
                        bob->recieveDamage(25); // The method is now called and bob takes 25 damage
                        bob->recieveDamage(100);

                        //Now you can check to see if the goblin is dead or not
                        if (bob->health <= 0) {
                            //If the goblin is dead you can delete the goblin now
                            delete bob;
                        }
                        
                        return 0;
                    }
